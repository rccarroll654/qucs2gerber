# qucs2gerber Package

This is a tool that generates Gerber code from a Qucs schematic file.

Example Qucs schematic:
![Qucs Example](/images/qucs_example.png)

Example qucs2gerber usage:

**python3 -m qucs2gerber -s demo.sch -o output.grb**
